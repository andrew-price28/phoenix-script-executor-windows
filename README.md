## FAQ

**Is Phoenix Executor safe to use on my main Roblox account?**  
Any third-party executor involves risk. The safer route is to use alternate accounts and to take full responsibility for anything you do with the tool.

**How often does the auto-update feature work?**  
The updater checks for new builds every time Phoenix Executor starts. Compatibility updates usually arrive within 24-48 hours after a Roblox patch.

**Can I use this alongside other Roblox mods or injectors?**  
Using more than one injection tool at the same time can create conflicts. For the most reliable behavior, run Phoenix Executor on its own.

**What happens to my local scripts if I reinstall?**  
Scripts saved in `scripts/user/` are kept during normal updates, but a clean reinstall can remove them. Make sure to back them up regularly.

**Does Phoenix Executor store any personal data?**  
No personal information is collected or sent anywhere. Configuration data and scripts stay on your local machine.
